# Get-UsrQuota

## SYNOPSIS
Use \"default\" as the serverGUID to return \r\nuser settings for the current server you connected to.

## SYNTAX

### Get (Default)
```
Get-UsrQuota -AdminUrl <String> -AuthGuid <String> -ServerGuid <String> -UserGuid <String> [-ByUserName]
 [<CommonParameters>]
```

### GetViaIdentity
```
Get-UsrQuota -AdminUrl <String> -InputObject <ITitanApiIdentity> [-ByUserName] [<CommonParameters>]
```

## DESCRIPTION
Use \"default\" as the serverGUID to return \r\nuser settings for the current server you connected to.
