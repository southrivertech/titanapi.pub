# Invoke-DownloadFile

## SYNOPSIS
Download a file

## SYNTAX

### Download (Default)
```
Invoke-DownloadFile -AdminUrl <String> -FileId <String> -OutFile <String> [-FileType <String>] [-PassThru]
 [<CommonParameters>]
```

### DownloadViaIdentity
```
Invoke-DownloadFile -AdminUrl <String> -InputObject <ITitanApiIdentity> -OutFile <String> [-FileType <String>]
 [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Download a file
