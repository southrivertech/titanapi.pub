# Get-FileMyCloudList

## SYNOPSIS
Return files at the server level which will consist of a root folder that contains\r\n            all cloud folders defined at the server level

## SYNTAX

### Get (Default)
```
Get-FileMyCloudList -AdminUrl <String> -ServerGuid <String> [-FileId <String>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-FileMyCloudList -AdminUrl <String> -InputObject <ITitanApiIdentity> [-FileId <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Return files at the server level which will consist of a root folder that contains\r\n            all cloud folders defined at the server level
