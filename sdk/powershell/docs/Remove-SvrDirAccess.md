# Remove-SvrDirAccess

## SYNOPSIS
Delete a dir access entry at either the server, user or group level by specifying the proper ownerGUID

## SYNTAX

### Delete (Default)
```
Remove-SvrDirAccess -DirAccessGuid <String> -OwnerGuid <String> -ServerGuid <String> [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-SvrDirAccess -InputObject <ITitanApiIdentity> [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
